Projeto Analisador Automático De Logs Com Relatorio Word

Projeto que baseado nos dados gera um log em formato .word para melhor visualização dos dados contidos nos logs.

Sobre o Projeto

Este projeto foi desenvolvido como parte do módulo de Manipulação de Word do curso Python Automático. O objetivo principal era aprender a manipular dados e exportalos em formato .word, tornando a sua visualização mais profissional. Com esse script que desenvolvi se torna mais simples e dinamico criar arquivos .word usando Python.

Tecnologias Utilizadas
* Linguagem: Python 3.10
* Framework/Lib: pypdf

Como Executar o Projeto

Para rodar este projeto localmente, siga os passos abaixo:

baixe o arquivo main e use python como linguagem, tornando assim totalmente editavel e funcional o script.
